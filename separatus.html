Separatus

Framework designed to create masks in COCO JSON format using histogram pixel inference

We took images and annotations from the publically available iPATCH dataset and created a framework that automatically generates segmentation masks to be used in a training sequence. The dataset originally consisted of videos of boats, along with bounding boxes that isolated the location of the boats. Below is an example of ground truth bounding boxes next to predicted bounding boxes. 

With the current setup, the masks that we could create with our state of the art detection algorithm, MaskRCNN, was subpar. Below is an example of mask generation before using our framework. 

By supplying images of positive and negative examples of boats, we could build histograms that aided us in being able to make segmentation masks. These are the masks that we automatically generated by using these histograms. 

After incorporating these masks back into our dataset and retraining the model on a set of predefined weights, we were able to get much better masks.

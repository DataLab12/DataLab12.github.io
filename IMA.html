Tool: Web App wrapper for Deep Learning

Mask RCNN Web based application designed to accept an image over a network, run inferencing, and return detections to user in COCO JSON format

We built a MaskRCNN web application that accepts an image as input from a web browser, processes that image to detect objects, and then returns all detections in a COCO JSON file back to the user. We also walk through how to launch this application on a remote server, and then access that remote server from a local machine.

Object Localization and Identification in Maritime Videos

Motivation: Maritime imagery properties differ from consumer imagery. How applicable are state-of-art machine learning method to this domain for piracy detection? Approach: Proposed and deployed multiple solutions for high-confidence maritime object detection, small sample learning schema for maritime object identification given few samples, and maritime activity modeling.

Identifying Structures in Aerial Imagery

Motivation: How to utilize open source city data to improve the AI systems? Approach: Transform open source city data to machine vision training data; adapt and train deep network to identify structures of varying size in overhead imagery; adjust framework to work on edge, and deploy it on Nvidia Jetson TX2 mobile platform.

Sclalable Health Data Integration and Analysis

Motivation: Health data is unstructured, noisy, and incomplete. How can data science help unlock the information? Approach: Revisit state-of-art from data science perspective; utilize data reduction and network techniques to scale integration and analysis of biological and clinical data.

Scaling Social Balance Theory for Social and Content Network Analysis

Motivation: Social Networks model the community dynamics. How to describe and identify inequity and bias in that community? Approach: Propose innovative scalable graph processing algorithm that scales social balance theory to automatically quantify disbalance and to predict outcomes in social and content networks.

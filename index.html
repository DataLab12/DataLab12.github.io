Research Highlights

Object Localization and Identification in Maritime Domain

We have developed multiple algorithms and tools to locate and identify maritime objects from overhead imagery in real time with high recall, see maritime project central for more details.

Identifying Structures in Aerial Overhead Imagery

We train a deep learning model to identify structures of varying size in overhead imagery. We adapt the framework for Mobile Deep Learning, and deploy it on Nvidia Jetson TX2 mobile platform.

Sclalable Health Data Integration and Analysis

We use data reduction and network analysis techniques to scale integration and analysis of biological and clinical data.

Social Balance Theory for SNA

We have proposed and implemented algorithm for quantifying disbalance and predicting outcomes in social networks that scales social balance theory to large social networks.
